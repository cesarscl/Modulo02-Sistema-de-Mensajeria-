# Sistema de Mensajería

Proyecto estático desarrollado por Cesar Ormeño como práctica de maquetación con **HTML** y **CSS**, enfocado en el uso de **Flexbox** para construir una interfaz tipo sistema de mensajería.

## Acceso

https://cesarscl.github.io/


## Descripción

La página muestra una aplicación de mensajería con dos áreas principales:

- Un panel lateral con perfil de usuario, botón para iniciar un nuevo chat y listado de contactos.
- Un área principal con una imagen destacada y un mensaje inicial para comenzar una nueva conversación.

El diseño utiliza imágenes locales, distribución flexible, botones, estados `hover` y `focus`, además de estilos personalizados para simular iconos de mensaje y menú.

## Tecnologías Utilizadas

- HTML5
- CSS3
- Flexbox

## Características

- Layout principal construido con Flexbox.
- Panel de contactos fijo al costado izquierdo.
- Sección principal centrada vertical y horizontalmente.
- Imágenes de perfil circulares.
- Botón principal con estado `hover` y `focus`.
- Iconos creados con CSS puro.
